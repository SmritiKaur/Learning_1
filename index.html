Cryptography

Cryptography is an art of writing or solving codes.It is a technique to provide message confidentially and involves the process of encryption and decryption where encryption is the process of encoding a message in such a way that only authorized parties can access it while decryption is the process of transforming encrypted information known as ciphertext back to plaintext or the original data.

Mathematics In Cryptography

Any maths that is simple to compute but hard to reverse like multiplication or factoring can be used in cryptography.

  • Number Theory
  • Combinatorics
  • Probability Theory
  • Statistical Analysis
  • Information Theory
  • Real Analysis and Calculus(occassionally)
  • Fourier Analysis
  • Complex Analysis (less often)

Caesar Cipher

The Caesar Cipher is one of the earliest known and simplest ciphers. It is a type of substitution cipher in which each letter in the plaintext is shifted a certain number of places down the alphabet.
The problem with shift cipher is that it is fairly easy to break esp. using a computer just by analyzing which letter appear most frequently in the encoded message. So, to solve this problem a more powerful method of encryption known as the RSA Algorithm that is very difficult for even the most powerful supercomputer to break was invented by 3 professors of MIT named Ron Rivest, Adi Shamir and Leonard Adleman.

RSA Algorithm

RSA is a consequence of Fermat's Little Theorem which states that if p is a prime number then for any integer a, the number a^(p-a) is an integer multiple of p i.e., a^p = a(mod p)

So, how Fermat's Little Theorem is used in RSA. RSA uses key encryption i.e., it uses two keys - Public key and the Private key. We use Fermat's Little Theorem in th encryption and decryption scheme to calculate the numbers e and d.

Function used for Encryption
(m^e)mod n = c

Function used for Decryption
(c^d)mod n = m

Private key - d

where m is the original message to be encrypted and c is the ciphertext.

RSA As A One Way Trapdoor Function

RSA is also known as the one way trapdoor function because we cannot undo the encryption function until we know the trapdoor and over here number n acts as a trapdoor because of the prime factorization it is very difficult to find n as it is easy to find the product of two prime numbers but it is very hard to take a very large product and factor it to find the two primes that compose it.
